# Screening picklist generator
## Description
Press the blue Play P (pipeline) toolbar button to run this pipeline. This pipeline runs the screening picklist generator in Python.

A single plate map is user-created to define numbered positions in a destination plate template were library compounds will be dispensed, at one or more concentrations, replicates, plus controls. As a third (LIB) worksheet the library table is added to the picklist generator Excel workbook. The SRC sheet describes from which columns of the library table to pull compound names, well and plate bar codes. The library table is threaded into the plate map template, using as many destination plates, as many needed to dispense the whole library. Here each library plate will be dispensed into multiple destination plates and some destination plates will get compounds from two library plates. Additional control list files required for this programming PlateWorks are generated by the script. Destination plate well, barcode, compound association lists, and plate maps for compounds and concentrations are also generated.
